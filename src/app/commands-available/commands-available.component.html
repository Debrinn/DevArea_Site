Le serveur dispose d'un bot créer pour ajouter de nouvelles fonctionnalitées lié au développement !

Vous pouvez utiliser toutes les commandes si dessous:

//help -> donne la liste des commandes disponible.
//ping -> donne la latence du bot en milli-secondes.
//meetup -> permet de mettre en place des conférence/regroupement vocal autour d'un sujet.
//start -> donne les près requis nécessaire pour débuté dans les langages disponibles.
//rank -> affiche votre rang, et votre xp sur le serveur.
//leaderboard -> affiche la liste des 5 premiers membres du serveur.
//devhelp -> mentionne les membres possédant le rôle Dev'Helper dans un channel d'entraide.
//ask -> affiche les informations nécessaire pour poser une question efficacement.
//mission -> permet de gérer vos propositions de missions.

creationMission -> cette commande se lance dans le channel missions-payantes pour créer une mission.

Toutes les commandes commencant par "//" doivent être utiliser dans les salons du serveur !
